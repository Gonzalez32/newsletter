Learn more about the Fantasy Football Newsletter and your league

🏈 Welcome to Fantasy Football Newsletter

This is your league's central hub for all things fantasy football! We've created this beautiful, modern website to keep track of your league's newsletters, standings, matchups, and player statistics.

📰 What We Do

  • Weekly Newsletters: Archive and display all your hilarious weekly newsletters
  • Team Standings: Real-time rankings and statistics for all teams
  • Matchup Tracking: Complete game history with scores and results
  • Player Management: Roster tracking and player statistics
  • League Dashboard: Comprehensive overview and quick stats

🎨 Features

📱 Responsive Design

Works perfectly on desktop, tablet, and mobile devices

🎨 Beautiful UI

Modern design with Bulma CSS framework

⚡ Fast Loading

Static site generation for optimal performance

🔒 Secure

HTTPS enabled with automatic SSL certificates

🛠 Tech Stack

  • Static Site Generator: Jekyll
  • Frontend Framework: Bulma CSS
  • Hosting: GitHub Pages
  • Domain: Custom domain with SSL
  • Data Format: JSON
  • JavaScript: Vanilla JS
  • Icons: Emoji and Font Awesome
  • Deployment: Automatic via Git

📝 How to Update Content

To update your league's content:

  1. Edit JSON files in assets/data/ to update team and league data
  2. Add newsletters by creating new files in _newsletters/ directory
  3. Commit and push changes to GitHub
  4. GitHub Pages will automatically rebuild and deploy

🤝 Contributing

This site is designed to be easily maintainable by league members. If you have suggestions for improvements or want to add new features, feel free to contribute!
